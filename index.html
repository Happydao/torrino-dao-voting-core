Peso Di Voto

Come viene calcolato il potere di voto degli NFT

Nel sistema di governance della DAO, il potere di voto dipende dagli NFT posseduti.

  • Ogni NFT della collezione Torrino DAO (Gen1) vale 0,9 punti di voto.
  • Ogni NFT della collezione Solnauta (Gen2) vale 0,1 punti di voto.

Il potere di voto totale e' la somma dei due valori calcolati sul wallet collegato.

La DAO dovrebbe adottare un'interfaccia di voto proprietaria con potere di voto pesato in base agli NFT detenuti nel wallet?

Codice pubblico e verificabile

Qui sotto vedi direttamente il blocco principale del codice usato per collegare Phantom e leggere i dati del wallet tramite backend. 

const RPC = process.env.HELIUS_RPC;

async function handleWalletNfts(requestUrl, res) {
  const walletAddress = requestUrl.searchParams.get("address");
  const assets = await getAllAssetsByOwner(walletAddress);
  const summary = summarizeWalletAssets(assets);
  sendJson(res, 200, summary);
}

async function connectWallet() {
  const provider = getPhantomProvider();

  if (!provider) {
    setStatus(getMissingWalletMessage());
    return;
  }

  const response = await window.solana.connect();
  const walletAddress = response.publicKey.toString();
  const responseData = await fetch(
    "/api/wallet-nfts?address=" + encodeURIComponent(walletAddress)
  );
  const walletData = await responseData.json();
  updateDashboard(walletAddress, walletData);
}
  • La connessione al wallet avviene con window.solana.connect().
  • Il browser legge solo l'indirizzo pubblico del wallet.
  • Non sono presenti sendTransaction o signAndSendTransaction.
  • L'RPC Helius e' sul backend e viene letto tramite process.env.HELIUS_RPC.
