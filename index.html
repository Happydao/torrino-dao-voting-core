Peso Di Voto

Come viene calcolato il potere di voto degli NFT

Nel sistema di governance della DAO, il potere di voto dipende dagli NFT posseduti.

  • Ogni NFT della collezione Torrino DAO (Gen1) vale 0,9 punti di voto.
  • Ogni NFT della collezione Solnauta (Gen2) vale 0,1 punti di voto.

Il potere di voto totale e' la somma dei due valori calcolati sul wallet collegato.

Codice pubblico e verificabile

Qui sotto vedi direttamente il blocco principale del codice usato per collegare Phantom e leggere i dati pubblici del wallet. 

const connection = new solanaWeb3.Connection(RPC_URL, "confirmed");

async function connectWallet() {
  const provider = getPhantomProvider();

  if (!provider) {
    setStatus(getMissingWalletMessage());
    return;
  }

  const response = await window.solana.connect();
  const walletAddress = response.publicKey.toString();
  const counts = await getVotingCounts(walletAddress);
  updateDashboard(walletAddress, counts);
}

async function getOwnedNftMints(walletAddress) {
  const ownerPublicKey = new solanaWeb3.PublicKey(walletAddress);
  const result = await connection.getParsedTokenAccountsByOwner(
    ownerPublicKey,
    { programId: TOKEN_PROGRAM_ID },
    "confirmed"
  );

  return result.value
    .filter((account) => {
      const tokenAmount = account.account.data.parsed.info.tokenAmount;
      return tokenAmount.amount === "1" && tokenAmount.decimals === 0;
    })
    .map((account) => account.account.data.parsed.info.mint);
}
  • La connessione al wallet avviene con window.solana.connect().
  • Il codice legge solo l'indirizzo pubblico e gli NFT del wallet.
  • Non sono presenti sendTransaction o signAndSendTransaction.
  • Il potere di voto viene solo calcolato nel browser partendo dai dati pubblici della blockchain.
