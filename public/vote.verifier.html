How To Verify

Public verification in 3 steps

Step 1 Open the public CSV Go to the GitHub data folder and open the CSV file for the proposal you want to inspect.
Step 2 Copy one vote row In the same row, copy the wallet, the signed message and the signature exactly as they appear.
Step 3 Run the verifier Paste the three values below and click verify. You will get TRUE if the row is authentic, or FALSE if something was altered.
