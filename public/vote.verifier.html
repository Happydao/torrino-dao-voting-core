How To Verify

Public verification in 4 steps

Step 1 Open the public CSV Open the GitHub CSV archive and choose the proposal file you want to inspect. Open CSV archive
Step 2 Rebuild the proposal hash manually In the CSV metadata and proposal block, copy proposal_payload_hash, proposal_title, proposal_description and the proposal options exactly as written.
Step 3 Copy one vote row In the same vote row, copy wallet, signed_message and signature exactly as they appear.
Step 4 Verify the vote row Paste the vote data and the same proposal_payload_hash. The verifier checks the wallet signature and confirms that the hash inside the signed message is the same hash you rebuilt manually.
