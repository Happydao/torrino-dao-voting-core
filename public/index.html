NFT Voting Weight

How voting power is calculated

Voting power depends entirely on the NFTs held in the connected wallet.

  • Each Torrino DAO NFT (Gen1) is worth 0.9 voting points.
  • Each Solnauta NFT (Gen2) is worth 0.1 voting points.

The total shown in the proposal and the results is the sum of these weights.

Transparency

Governance is fully transparent. The source code and voting results are public on GitHub.

  • The platform source code is public.
  • Voting results are public.
  • Votes are recorded off-chain.
  • CSV files are published automatically on GitHub.
  • Anyone can verify the results publicly.

During voting, the results file is updated automatically and can be verified publicly.
