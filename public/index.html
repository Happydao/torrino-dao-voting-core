NFT Voting Weight

How voting power is calculated

Voting power depends entirely on the NFTs held in the connected wallet.

  • The full Gen1 collection always represents 90% of the treasury voting power.
  • The full Gen2 collection always represents 10% of the treasury voting power.
  • Each Gen1 NFT therefore carries 0.180000 voting power.
  • Each Gen2 NFT therefore carries about 0.011261 voting power.

This means the maximum combined voting power is 100, with Gen1 fixed at 90 and Gen2 fixed at 10.

Transparency

Governance is fully transparent. The source code and voting results are public on GitHub.

  • The platform source code is public.
  • Voting results are saved in a CSV file.
  • CSV files are published automatically on GitHub.
  • Votes are publicly verifiable and tamper-resistant.
  • Each vote is protected by a cryptographic signature, proving it was cast by the wallet and cannot be altered.

During voting, the results file is updated on GitHub every 10 minutes.
